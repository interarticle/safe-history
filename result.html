Learn to spot the telltale signs of a secure site. At minimum, it should have a URL that starts with https://, and it should have a padlock icon in the browser frame – NOT on the site's page. These show the site is protected with Secure Sockets Layer (SSL), a cryptographic protocol that provides communication security.
